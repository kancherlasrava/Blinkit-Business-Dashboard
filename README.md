# Blinkit-Business-Dashboard
Power BI dashboard analyzing Blinkit sales, outlet performance, and customer insights using data visualization and DAX.

##  Project Overview
This project analyzes Blinkit’s sales performance, outlet efficiency, and customer behavior using Power BI. The goal is to generate actionable business insights and support data-driven decision-making.

---

##  Objectives
- Analyze overall sales performance
- Compare outlet performance across different city tiers
- Understand customer preferences and product trends
- Identify underperforming areas for improvement

---

## Tools Used
- Power BI (Power Query for data cleaning, Data Modeling, DAX, Visualization)
- Excel (Initial data review / basic preprocessing)

---

##  Key Insights
- Tier 3 cities generate the highest sales, followed by Tier 2 and Tier 1
- Medium-sized outlets contribute the most to overall revenue
- Sales peaked around 2018 and showed fluctuations afterward
- Item visibility does not always result in higher sales

---

## Files Included
- `Blinkit_Business_Dashboard.pbix` – Power BI dashboard file  
- `Blinkit_Grocery_Dataset.xlsx` – Dataset used for analysis  

---

## Dashboard Features
- KPI cards for total sales, average sales, and ratings  
- Sales analysis by item type, outlet size, and location  
- Trend analysis using line charts  
- Scatter plot for visibility vs sales relationship  
- Interactive filters for better analysis  

---

##  Conclusion
The analysis shows that Blinkit performs strongly in Tier 2 and Tier 3 cities and through medium-sized outlets. However, inconsistent sales trends and weak visibility impact highlight areas for strategic improvement.
